# 简介
```markdown
"《借过一下》是由周深演唱、周以力作曲制作、唐恬作词的《庆余年第二季》片尾曲。音乐团队以男主范闲的人物为创作核心： 有节奏感的曲是男主的力量，潇洒通透的词是男主的态度、极具少年感的演唱是男主的光芒。 对于范闲这个充满魅力、亦古亦今的人物，词作者把古风和现代口语在同首歌内碰撞，来呈现范闲穿越的身份、破局又破格的行事风格。周深在演唱里代入了范闲的\"潇洒\"、\"侠气\"和\"聪明\"。主歌高低调两轨演唱相叠加，给歌曲带来了惊艳又不一样的听感。 \"市井和朝堂里有无数看不见的规矩和道理，前辈和大人们说得都对，但麻烦借过一下，我依然要赶赴我的月亮。\" 希望当音乐起时，范闲在剧中，亦在现世当下。 白衣过泥潭，世故里浪漫 知心有所爱，不更改 人见过明暗，若天真犹在 当披上月光，走夜晚"
```
