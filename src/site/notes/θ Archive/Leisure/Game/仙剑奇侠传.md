# 剧情
"《仙剑奇侠传》是由大宇资讯制作的一款国产单机中文角色扮演游戏。该游戏是《仙剑奇侠传》系列的第一部作品，主题为“宿命”。 游戏讲述了平凡的客栈小伙计李逍遥因机缘巧合结识了女娲族后裔赵灵儿。赵灵儿所居住的仙灵岛意外遭人袭击，李逍遥遂担任起护送灵儿前往苗疆寻母的重任。在旅程中，李逍遥先后结识了林月如和阿奴。在赵灵儿帮助苗人祈雨解旱、惩奸除恶的过程中，赵灵儿的身世之谜以及她母亲的下落也逐渐揭开。最终，李逍遥、赵灵儿等人与拜月教主展开了一场决定命运的战斗。 《仙剑奇侠传一》以其丰富的剧情、深刻的人物刻画和独特的中国神话背景，成为了中文游戏界的经典之作，并获得了多项游戏奖项。"
