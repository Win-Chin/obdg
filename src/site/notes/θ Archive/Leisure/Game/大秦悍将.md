# 剧情
"游戏独创“双主角”概念，机甲被击毁后驾驶员可藉由弹射装置逃出生天，只要短时间内躲过敌方的追杀便可卷土重来。而占优的一方则力求赶尽杀绝，同时还须提防其他机甲的偷袭，实战中往往生死悬于一线，战斗更趋紧张激烈。 《大秦悍将》提供了多种游戏模式，包括死亡竞赛、团队作战、占山为王等，基本上聚合了时下流行的各种游戏模式之精华。更推出全新的“抓俘虏”、“救俘虏”等模式，击毁敌方机甲后可以用捕人器抓住驾驶员扔进监狱，被俘者将捱过30秒种的铁窗生涯。当然，这其间有机会得到队友的营救，为了获得团队的最终胜利，每位玩家都会想方设法救出己方的队友。游戏中玩家可以淋漓尽致地体会团队作战之力量、战场友情的珍贵。 玩家鏖战的地图包括皇城、鬼城、天坛、敦煌、少林寺等民族特色浓郁的名胜古迹；罗马竞技场、复活节岛等充满异国风情的著名景点；太空站、月球等光怪陆离的未来世界。这些场景无不惟妙惟肖，宛若身临其境。玩家将自由穿梭于秦皇汉武、埃及玛雅的千年时空，足不出户而神游万里，凭借超级火力，决战紫禁之巅！ 游戏另一个亮点是纯中文界面和语音，在游戏中可以用你的母语指挥战友冲锋陷阵，手刃敌人；支持实时通信，尽显中文协同作战的威力。甚至还体贴的预设了通讯快捷键，激战中轻轻点几下键盘，便可发出一个个真人配音的指令，如“冲锋”，“就在这里埋伏”，“我被击中了需要支援”等等，让你在百忙之中仍能指挥若定，最大限度发挥团队的战力。 《大秦悍将》除了上述的一些鲜明的特色，其3D引擎流畅自如，游戏画面华丽细致，音乐音效雄浑激越，亦为业界与玩家所深度认同。欢乐亿派在游戏推出前于网上连续发布测试版，认真听取了上千名玩家的意见与建议，并潜心加以改进。"
