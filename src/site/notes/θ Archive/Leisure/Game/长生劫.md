# 剧情
"点烛吹灯惊鬼冤，八字不硬莫敢前。千载迷局谁人破？机变无形在此间。 细思极恐的盗墓剧情，烧脑又上瘾的机关破解玩法，还有N种场景互动式的陷阱解谜，下墓累了还能佛系钓鱼养家，跟黑商斗智斗勇。精心炮制的盗墓题材，生存解谜单机《长生劫》，等待你来挑战！ 【故事背景】 传说上古三皇，铸成长生之法。后人皆向往，欲求之而不得。直至汉高祖刘邦之孙，淮南王刘安，踏遍千山，访寻先王墓陵，得长生之法，遗留长生印流传于世间。后长生印隐匿于历代帝皇手中，下落渐渐不得而知。 千年后，吕烽等人受托，携带淮南图残卷，重启长生印探寻之路。不料，在淇河大墓探寻中因滔天巨浪与队员走散。至此，吕烽便卷入一场先王谋划千年的阴谋之中…… 【游戏特色】 1.抽丝剥茧，千载迷局扑朔迷离 上古三皇，成就长生伟业流传于世，然而长生岂无代价？秦三十七年，擅长生术的鬼谷全族遭诛，始皇又是否真的驾崩？至汉，淮南王刘安得长生之法，鸡犬升天，为何却未见其长生？这一切背后隐藏着长生印的秘密沉寂千载，直至千年后，故事在你身上重新开启，然而你踏出第一步开始便已入局。这场阴谋，最后真正的赢家会是谁？ 2.机变无形，生死抉择存乎一念 九州五岳连锁机关，八宝盒华容道解谜……这里的机关变化万千。你将跟随剧情，步步深入，数十种烧脑机关等待你一一破解。而且你要知道，如果你无法解开这些机关，你也将永远止步当前！越是危机重重，越是要沉着应对，只有真正的摸金校尉强者才能最终接触到藏在墓下千年的真相。 3.危机四伏，墓下千险步步为营 淇河大墓、商洛山古墓、秦始皇陵、南洋古墓、上古魔窟，千年古墓风貌各异的同时危机四伏，稍有不慎便深陷迷阵。那平坦的墓道可能隐藏着致命陷阱，那火光照不亮的漆黑一角游荡着的是陪葬的孤魂，那一个个棺椁推开之后谁能知是祸是福……食物已经见底，火把也即将燃烧殆尽，你还要继续往前吗？ 4.运筹帷幄，苦心经营倒斗小队 在这里，你将面临许多真实的抉择。物物交易的社会，你将怎样与黑商斗智斗勇，以最优的策略换取珍稀的下墓物资？囊中羞涩的时候，你该如何选择招募的伙伴？下墓能携带的物资有限，你可没法都带走，必须做出选择！墓里琳琅满目的珍宝与生存必备的干粮，你又要做出怎样的取舍呢？只有苦心经营，精打细算，才能在墓下世界生存。"
